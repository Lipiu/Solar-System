# 🌌 Animated Solar System

An interactive web application that simulates the Solar System with animated planetary orbits around the Sun. The project combines educational content with engaging visuals, allowing users to explore the planets while enjoying an interactive experience.

## ✨ Features

* 🌍 Smooth animations of planets orbiting the Sun
* 📖 Dedicated information page with interesting facts about each planet
* 🎵 Built-in background music controls:

  * Play
  * Pause
  * Mute/Unmute
